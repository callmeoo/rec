在 `src/components/VehicleDetail/VehicleDetailPage.jsx` 中：

```javascript
images: [
  { 
    url: '你的图片URL1', 
    type: 'exterior', 
    alt: '车辆外观' 
  },
  { 
    url: '你的图片URL2', 
    type: 'interior', 
    alt: '车辆内饰' 
  },
  // ... 更多图片
]
```

### 方法2：对接真实API

```javascript
const loadVehicleData = async () => {
  setLoading(true);
  try {
    // 调用真实API
    const response = await fetch(`/api/vehicles/${vehicleId}`);
    const data = await response.json();
    setVehicle(data);
  } catch (error) {
    console.error('加载失败:', error);
  } finally {
    setLoading(false);
  }
};
```

## 🎨 图片要求

### 推荐尺寸
- **主图**：800x600 像素
- **缩略图**：400x300 像素
- **相似推荐**：400x300 像素

### 格式要求
- 支持：JPG、PNG、WebP
- 推荐：WebP（体积小，质量高）

### 质量要求
- 清晰度：高清
- 压缩：适度压缩（保持质量）
- 大小：单张 < 500KB

## 💡 优化建议

### 1. 懒加载
已实现图片懒加载，提升页面加载速度

### 2. 响应式图片
根据设备尺寸加载不同大小的图片：
```html
<img 
  srcset="
    small.jpg 400w,
    medium.jpg 800w,
    large.jpg 1200w
  "
  sizes="(max-width: 768px) 400px, 800px"
/>
```

### 3. WebP格式
优先使用WebP格式，降级到JPG：
```html
<picture>
  <source srcset="image.webp" type="image/webp">
  <img src="image.jpg" alt="车辆图片">
</picture>
```
