## 🐛 常见问题

### Q1: 点击车辆卡片没有反应？
A: 检查浏览器控制台是否有错误。确保已经正确传递了`onVehicleClick`属性。

### Q2: 详情页显示空白？
A: 检查`vehicleId`是否正确传递。查看控制台是否有数据加载错误。

### Q3: 利润计算器不更新？
A: 检查滑块是否可以拖动。查看控制台是否有计算错误。

### Q4: 图片不显示？
A: 当前使用的是占位图片。实际使用时需要替换为真实图片URL。

### Q5: AI解析内容是模拟的吗？
A: 是的，当前是模拟数据。实际使用时需要对接真实的车况报告API。

打开浏览器控制台（F12），查看埋点事件：

### 应该看到的埋点事件：

1. **进入详情页**
   ```javascript
   detail_page_view: {
     vehicle_id: "V001",
     source: "recommendation",
     device_type: "pc"
   }
   ```

2. **使用利润计算器**
   ```javascript
   profit_calculator_use: {
     vehicle_id: "V001",
     bid_amount: 380000,
     profit: 25000
   }
   ```

3. **展开AI解析**
   ```javascript
   ai_analysis_toggle: {
     vehicle_id: "V001",
     action: "open"
   }
   ```

4. **查看市场行情**
   ```javascript
   market_data_view: {
     vehicle_id: "V001"
   }
   ```

5. **点击出价按钮**
   ```javascript
   bid_button_click: {
     vehicle_id: "V001",
     current_price: 350000
   }
   ```
