# 车辆详情页图片说明

## 概述

车辆详情页已实现为一个弹窗组件，直接显示您提供的车辆详情页图片。

## 图片放置位置

请将您提供的车辆详情页图片保存到以下位置：

```
public/vehicle-detail-template.jpg
```

## 操作步骤

### 方法1：直接复制图片

1. 将您提供的车辆详情页图片重命名为 `vehicle-detail-template.jpg`
2. 将图片复制到项目的 `public` 文件夹中
3. 刷新浏览器即可看到效果

### 方法2：使用命令行

```bash
# 假设您的图片在下载文件夹中
cp ~/Downloads/车辆详情页.jpg public/vehicle-detail-template.jpg
```

## 支持的图片格式

- JPG/JPEG（推荐）
- PNG
- WebP

如果使用其他格式，请修改文件名：
- PNG格式：`vehicle-detail-template.png`
- WebP格式：`vehicle-detail-template.webp`

并相应修改 `src/components/VehicleDetail/SimpleVehicleDetailPage.jsx` 中的图片路径。

## 功能说明

### 触发方式

车辆详情弹窗会在以下情况下打开：

1. **PC端推荐页面**：点击任意车辆卡片
2. **移动端推荐页面**：点击任意车辆卡片
3. **平台周转排行**：点击车型列表中的"查看详情"按钮

### 弹窗特性

- ✅ 模态弹窗，覆盖在当前页面上
- ✅ 半透明黑色背景遮罩
- ✅ 响应式设计，自适应屏幕大小
- ✅ 可滚动查看完整图片
- ✅ 点击"关闭"按钮或按 ESC 键关闭

### 弹窗布局

```
┌─────────────────────────────────────┐
│  车辆详情                      [X]  │ ← 头部
├─────────────────────────────────────┤
│                                     │
│  ┌───────────────────────────────┐ │
│  │                               │ │
│  │    车辆详情页图片             │ │ ← 内容区域（可滚动）
│  │                               │ │
│  └───────────────────────────────┘ │
│                                     │
├─────────────────────────────────────┤
│                          [关闭]     │ ← 底部
└─────────────────────────────────────┘
```

## 图片要求

### 推荐尺寸

- 宽度：800-1200px
- 高度：根据内容自适应
- 分辨率：72-150 DPI

### 文件大小

- 建议：< 500KB
- 最大：< 2MB

### 图片质量

- JPG质量：80-90%
- PNG：使用压缩工具优化

### 常见修改

#### 修改图片路径

```javascript

// 修改为其他路径

```

## 完整流程示例

### 用户点击车辆卡片

```
用户操作：点击车辆卡片
    ↓
触发：onVehicleClick(vehicleId)
    ↓
App.jsx：setSelectedVehicleId(vehicleId)
    ↓
渲染：SimpleVehicleDetailPage 组件
    ↓
显示：车辆详情弹窗（包含图片）
    ↓
用户操作：点击关闭按钮
    ↓
触发：onClose()
    ↓
App.jsx：setSelectedVehicleId(null)
    ↓
关闭：车辆详情弹窗
```

## 注意事项

1. **图片路径**：图片必须放在 `public` 文件夹中，这样才能通过 `/` 路径访问
2. **文件名**：确保文件名完全匹配（区分大小写）
3. **浏览器缓存**：如果更新了图片但没有显示，请清除浏览器缓存或强制刷新（Ctrl+F5）
4. **开发服务器**：修改 public 文件夹中的文件后，可能需要重启开发服务器

## 后续优化建议

```javascript
useEffect(() => {

  img.src = '/vehicle-detail-template.jpg';
}, []);
```

### 3. 添加加载动画

```javascript
const [imageLoading, setImageLoading] = useState(true);

<img
  src="/vehicle-detail-template.jpg"
  onLoad={() => setImageLoading(false)}
/>
{imageLoading && <div>加载中...</div>}
```
