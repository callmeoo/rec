# 为您推荐模块演示指南

## 🚀 快速开始

项目已启动，访问地址：**http://localhost:5173/**

## 📋 演示步骤

### 1️⃣ 客户端 - 为您推荐页面

**操作步骤**:
1. 打开浏览器访问 http://localhost:5173/
2. 默认进入"猜你喜欢"页面
3. 查看推荐车辆列表（12辆车）

**功能演示**:
- ✨ **促销标签**: 注意车辆卡片左上角的标签
  - 🔴 红色"围观人数"：浏览人数≥100（如：特斯拉Model 3、理想L8）
  - 🟠 橙色"多人意向"：出价人数>3（如：奔驰E300L、比亚迪汉）
  - 🟢 绿色"快速周转"：周转分>70（大部分车辆）
  
- 💰 **价格信息**: 查看起拍价和当前价
  
- 📊 **互动数据**: 底部显示
  - 浏览人数（UV）
  - 出价次数
  - 出价人数
  
- 🎯 **推荐理由**: 卡片底部蓝色文字
  - "根据您的浏览记录推荐"
  - "您浏览过XX品牌"
  - "热门拍卖中"
  - "低里程精品车"
  - "准新车推荐"

- 🔍 **筛选功能**: 点击右上角"筛选"按钮
  - 价格区间滑块
  - 品牌多选
  - 车龄多选
  - 里程滑块
  
- 🔄 **换一批**: 点击"换一批"按钮刷新推荐

- ❤️ **收藏功能**: 点击卡片右上角心形图标

- ⚠️ **异常处理**: 
  - 点击车辆卡片会检查状态
  - 如果车辆已下架或竞价结束，会弹出提示

### 2️⃣ 管理后台 - 推荐算法验证

**操作步骤**:
1. 点击左侧菜单"推荐管理"
2. 点击子菜单"推荐算法验证"
3. 输入买家信息（支持多种方式）：
   - 买家ID：`B001` 或 `B002` 或 `B003`
   - 买家姓名：`张三` 或 `李四` 或 `王五`
   - 手机号：`13812345678` 或 `13956781234` 或 `13690129876`
4. 点击"查询"按钮

**功能演示**:
- 👤 **买家信息**: 顶部显示买家ID、姓名、电话、地区
  
- 📊 **买家画像**: 彩色卡片展示（新功能）
  - 🌍 **主打国别**: 日系(80%)、德系(15%)等
  - 🏷️ **主打品牌**: 大众(60%)、丰田(20%)等
  - 🚗 **主打车系**: 大众迈腾(55%)、丰田凯美瑞(30%)等
  - 💰 **主打价位段**: 0-3万(35%)、3-5万(28%)等
  - 📅 **主打车龄**: 4-6年(61%)、2-4年(25%)等
  - 每个维度都有进度条可视化显示百分比
  
- 📋 **推荐列表**: 固定显示30条推荐车源
  
- 🔢 **排序**: 第一列显示推荐位序号（1-30）
  - 前三名有特殊的圆形徽章
  
- 🚗 **推荐车源**: 显示品牌车系、车型、车牌
  
- 📍 **基本参数**: 
  - 国别（德国、日本、美国）
  - 所在地（北京、上海、广州等）
  - 表显里程
  - 车龄
  - 车况评分
  
- 💵 **起拍价/状态**: 
  - 当前价格（红色大字）
  - 起拍价（灰色小字）
  - 倒计时（橙色）
  
- 📈 **互动性**: 
  - 浏览次数/人数
  - 出价次数/人数
  
- 🏷️ **推荐逻辑**: 彩色标签显示
  - 🔵 蓝色"画像匹配-主打车型"
  - 🟢 绿色"候选车源-同车型高频成交"
  - ⚪ 灰色"兜底-平台热销"
  - 显示候选集编号（1/2/3）

**测试数据**:
- B001 / 张三 / 13812345678：主打日系(80%)、大众(60%)、迈腾(55%)、0-3万(35%)、4-6年(61%)
- B002 / 李四 / 13956781234：主打德系(70%)、奔驰(45%)、E级(40%)、15-20万(45%)、2-4年(55%)
- B003 / 王五 / 13690129876：主打日系(65%)、丰田(50%)、凯美瑞(45%)、5-8万(40%)、4-6年(50%)

**查询方式演示**:
1. 按买家ID查询：输入 `B001`
2. 按姓名查询：输入 `张三`
3. 按手机号查询：输入 `13812345678`
4. 所有方式都能查到同一个买家的信息和画像

### 3️⃣ 管理后台 - 平台周转排行

**操作步骤**:
1. 点击左侧菜单"推荐管理"
2. 点击子菜单"平台周转排行"
3. 查看车系排行榜

**功能演示**:
- 📊 **统计概览**: 顶部四个数据卡片
  - 车系总数：10
  - 最高周转分：显示第一名的分数
  - 平均成交率：所有车系的平均值
  - 平均耗时：所有车系的平均值
  
- 🏆 **排行榜**: 
  - 前三名有特殊徽章
    - 🥇 第1名：金色
    - 🥈 第2名：银色
    - 🥉 第3名：铜色
  - 其他名次：灰色圆形徽章
  
- 📋 **车系信息**: 
  - 车系名称（如：奔驰E级、宝马5系）
  - 平均成交率（百分比）
  - 成交耗时中位数（小时）
  - 综合周转分（彩色标签）
  
- 🎨 **周转分颜色**: 
  - 🟢 绿色：≥80分（优秀）
  - 🔵 蓝色：≥70分（良好）
  - 🟠 橙色：≥60分（一般）
  - 🔴 红色：<60分（较差）
  
- 📂 **展开车型**: 
  - 点击车系名称或"展开"按钮
  - 显示该车系下的3个车型
  - 每个车型显示：
    - 车型名称（如：奔驰E级 2020款）
    - 成交率
    - 成交耗时
    - 周转分（彩色数字）
    - 在拍数量（蓝色徽章）
  
- 🔗 **查看详情**: 
  - 点击车型行的"查看详情"按钮
  - 在新窗口打开车辆详情页（模拟）

## ❓ 常见问题

**Q: 为什么有些车辆没有促销标签？**
A: 促销标签有严格的条件：
- 围观人数：浏览UV ≥ 100
- 多人意向：出价人数 > 3
- 快速周转：周转分 > 70
只有满足其中一个条件才会显示标签。

**Q: 推荐算法验证页面为什么固定30条？**
A: 根据PRD要求，管理后台验证功能固定展示30条推荐车源，用于运营人员验证算法准确性。

**Q: 周转分是如何计算的？**
A: 周转分 = 成交率评分 × 0.7 + 耗时评分 × 0.3

**Q: 如何对接真实API？**
A: 修改以下文件中的数据获取逻辑：
- `src/components/CarRecommendation/recommendationEngine.js`
- `src/components/Admin/BuyerRecommendationVerify.jsx`
- `src/components/Admin/TurnoverRanking.jsx`
