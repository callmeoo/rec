# 推荐模块UI修改说明

## 修改概述

根据用户需求，对"为您推荐"模块进行了全面的UI和功能调整，包括PC端和移动端的推荐页面，以及管理后台的推荐算法验证页面。

## 详细修改内容

### 1. PC端推荐页面 (RecommendationPage.jsx)

#### 1.1 头部修改
- ✅ **去掉筛选按钮**：移除了"筛选"按钮及相关功能
- ✅ **去掉换一批按钮**：移除了"换一批"按钮

#### 1.2 车辆卡片修改 (CarCard.jsx)

**促销标签调整：**
- ✅ **去掉"围观人数"标签**：不再显示红色的围观人数标签
- ✅ **保留"周转快"标签**：当周转分 > 70 时显示绿色背景的"周转快"标签
- ✅ **"多人意向"标签**：当出价人数 > 0 时显示绿色背景的"多人意向"标签（替代原来的"X人出价"文字）

**布局调整：**
- ✅ **收藏按钮位置**：从图片右上角移至卡片右下角
- ✅ **车况评分显示**：在图片右上角显示车况评分，格式为"85B"、"90A"等
  - 90分以上：A级
  - 80-89分：B级
  - 70-79分：C级
  - 70分以下：D级

**其他调整：**
- ✅ **去掉车辆数量提示**：移除了"为您找到 X 辆推荐车辆"的提示文字

#### 1.3 车况评分等级规则
```javascript
90分以上 → A级（蓝色背景）
80-89分 → B级（蓝色背景）
70-79分 → C级（蓝色背景）
70分以下 → D级（蓝色背景）
```

### 2. 移动端推荐页面 (MobileRecommendationPage.jsx)

#### 2.1 头部修改
- ✅ **去掉换一批按钮**：移除了头部的"换一批"按钮

#### 2.2 车辆卡片修改 (MobileCarCard)

**促销标签调整：**
- ✅ **增加"多人意向"标签**：当出价人数 > 0 时显示绿色背景的"多人意向"标签
- ✅ **保留"周转快"标签**：当周转分 > 70 时显示绿色背景的"周转快"标签

**布局调整：**
- ✅ **收藏按钮位置**：从图片右上角移至卡片右下角
- ✅ **车况评分显示**：在图片右上角显示车况评分，格式为"85B"、"90A"等

**其他调整：**
- ✅ **去掉车辆数量提示**：移除了"为您找到 X 辆推荐车辆"的提示文字

### 3. 推荐算法验证页面 (BuyerRecommendationVerify.jsx)

#### 3.1 推荐列表修改
- ✅ **所在地显示优化**：只显示城市名称（如：北京、上海），去掉国别信息（德国、日本、美国）
- ✅ **推荐数量调整**：从30台车调整为16台车

#### 3.2 数据生成调整
```javascript
// 修改前：30台车，包含国别信息
Array.from({ length: 30 }, ...)

// 修改后：16台车，只显示城市
Array.from({ length: 16 }, ...)
```

## 标签显示逻辑

### PC端和移动端

1. **多人意向标签**（绿色）
   - 条件：`bidderCount > 0`
   - 显示：绿色背景，白色文字
   - 文本："多人意向"

2. **周转快标签**（绿色）
   - 条件：`turnoverScore > 70`
   - 显示：绿色背景，白色文字
   - 文本："周转快"

3. **围观人数**
   - 始终显示（如果有 viewUV 数据）
   - 格式："X人围观"
   - 样式：灰色文字
