## 🔍 问题排查步骤

### 步骤1：确认访问地址

请尝试以下地址（按顺序）：

1. **主要地址**：
   ```
   http://localhost:5173
   ```

2. **备用地址**：
   ```
   http://127.0.0.1:5173
   ```

3. **网络地址**：
   ```
   http://192.168.28.52:5173
   ```

### 步骤2：清除浏览器缓存

1. 打开浏览器
2. 按 `Ctrl + Shift + Delete`（Windows）或 `Cmd + Shift + Delete`（Mac）
3. 选择"清除缓存"
4. 刷新页面（`Ctrl + F5` 或 `Cmd + Shift + R`）

### 步骤3：使用无痕模式

1. Chrome：`Ctrl + Shift + N`（Windows）或 `Cmd + Shift + N`（Mac）
2. Firefox：`Ctrl + Shift + P`
3. Edge：`Ctrl + Shift + N`
4. 在无痕窗口中访问：`http://localhost:5173`

### 步骤4：检查浏览器控制台

1. 打开浏览器
2. 按 `F12` 打开开发者工具
3. 切换到 "Console" 标签
4. 访问 `http://localhost:5173`
5. 查看是否有红色错误信息

**常见错误及解决方案**：

#### 错误1：Module not found
```
Error: Cannot find module 'xxx'
```
**解决方案**：
```bash
npm install
```

#### 错误2：React is not defined
```
ReferenceError: React is not defined
```
**解决方案**：检查import语句

#### 错误3：Unexpected token
```
SyntaxError: Unexpected token
```
**解决方案**：检查代码语法错误

### 步骤5：检查网络标签

1. 在开发者工具中切换到 "Network" 标签
2. 刷新页面
3. 查看是否有失败的请求（红色）
4. 点击失败的请求查看详情

## 🛠️ 常见问题解决

### 问题1：页面空白

**可能原因**：
- React组件渲染错误
- JavaScript错误
- 路径问题

**解决方案**：
```bash
# 1. 停止服务器（Ctrl + C）
# 2. 重新安装依赖
npm install
# 3. 重新启动
npm run dev
```

### 问题2：样式不显示

**可能原因**：
- Tailwind CSS未正确配置
- CSS文件未导入

**解决方案**：
检查 `src/index.css` 是否在 `src/main.jsx` 中导入

### 问题3：Cannot GET /

**可能原因**：
- 路由配置问题
- 服务器未启动

**解决方案**：
```bash
# 确认服务器正在运行
npm run dev
```

## 🔧 重启服务器

如果以上方法都不行，尝试重启服务器：

### 方法1：在终端中

1. 按 `Ctrl + C` 停止服务器
2. 运行 `npm run dev` 重新启动

### 方法2：完全重启

```bash
# 1. 停止所有Node进程
# Windows
taskkill /F /IM node.exe

# Mac/Linux
killall node

# 2. 清除缓存
rm -rf node_modules/.vite

# 3. 重新启动
npm run dev
```

## 💡 临时解决方案

如果实在无法访问，可以尝试：

### 方案1：更改端口

编辑 `vite.config.js`：

```javascript
export default defineConfig({
  server: {
    port: 3000, // 改为3000端口
    host: true
  }
})
```

然后访问：`http://localhost:3000`
