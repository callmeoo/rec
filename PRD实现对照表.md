| 优先级1: 围观人数 | ✅ 已实现 | 浏览UV ≥ 100，红色标签 |

| 优先级3: 快速周转 | ✅ 已实现 | 周转分 > 70，绿色标签 |
| 只显示最高优先级标签 | ✅ 已实现 | `getPromotionTag()` 函数实现 |

**实现位置**: `src/components/CarRecommendation/CarCard.jsx` (第42-56行)

**代码片段**:
```javascript
const getPromotionTag = () => {
  // 优先级1: 围观人数
  if (car.viewUV && car.viewUV >= 100) {
    return { text: '围观人数', color: 'bg-red-500' };
  }
  // 优先级2: 多人意向（出价人数>3）
  if (car.bidderCount && car.bidderCount > 3) {
    return { text: '多人意向', color: 'bg-orange-500' };
  }
  // 优先级3: 快速周转（周转分>70）
  if (car.turnoverScore && car.turnoverScore > 70) {
    return { text: '快速周转', color: 'bg-green-500' };
  }
  return null;
};
```

const getScoreBadge = (score) => {
  const numScore = parseFloat(score);
  if (numScore >= 80) return 'bg-green-100 text-green-700';
  if (numScore >= 70) return 'bg-blue-100 text-blue-700';
  if (numScore >= 60) return 'bg-orange-100 text-orange-700';

};
