# 最简单的部署方法（5分钟完成）

## 方法一：使用 Netlify（最简单，拖拽即可）

### 步骤：

1. **打开浏览器**，访问：https://app.netlify.com/drop

2. **注册/登录账号**
   - 可以用 GitHub、GitLab、Email 注册
   - 完全免费

3. **拖拽部署**
   - 找到项目文件夹中的 `dist` 文件夹
   - 直接拖拽 `dist` 文件夹到网页上
   - 等待上传完成（约30秒-1分钟）

4. **获得外网链接**
   - 上传完成后，会自动生成一个链接
   - 格式类似：`https://random-name-123456.netlify.app`
   - 这就是您的外网访问地址！

5. **（可选）自定义域名**
   - 点击 "Site settings" → "Change site name"
   - 修改为您想要的名字
   - 例如：`https://your-car-auction.netlify.app`

## 方法二：使用 Vercel（需要命令行）

### 步骤：

1. **打开终端**，在项目目录运行：
```bash
npx vercel
```

2. **按照提示操作**：
   - 第一次使用会要求登录
   - 输入邮箱地址
   - 去邮箱点击验证链接
   - 回到终端，按回车继续

3. **回答几个问题**：
   - "Set up and deploy?"  → 按回车（Yes）
   - "Which scope?"  → 按回车（选择默认）
   - "Link to existing project?"  → 输入 `n` 按回车（No）
   - "What's your project's name?"  → 按回车（使用默认名字）
   - "In which directory is your code located?"  → 按回车（使用 ./）

4. **等待部署完成**
   - 会显示进度条
   - 完成后会显示外网访问链接
   - 格式类似：`https://your-project.vercel.app`

## 方法三：使用 GitHub Pages（如果项目在 GitHub 上）

### 步骤：

1. **确保项目已推送到 GitHub**

2. **在项目根目录创建文件**：`.github/workflows/deploy.yml`

3. **复制以下内容到文件中**：

\`\`\`yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [ main ]

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'
          
      - name: Install dependencies
        run: npm install
        
      - name: Build
        run: npm run build
        
      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: \${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./dist
\`\`\`

4. **推送代码到 GitHub**：
```bash
git add .
git commit -m "Add deploy workflow"
git push
```

5. **在 GitHub 仓库设置中启用 Pages**：
   - 进入仓库页面
   - 点击 "Settings"
   - 左侧菜单找到 "Pages"
   - Source 选择 "gh-pages" 分支
   - 保存

6. **等待几分钟，访问**：
   - `https://your-username.github.io/your-repo-name`

## 常见问题

### Q: 部署后图片显示不出来？
A: 确保 `public/vehicle-detail-template.jpg` 文件存在，然后重新构建和部署。

### Q: 可以修改网站名字吗？
A: 可以！
- Netlify: 在网站设置中修改
- Vercel: 在项目设置中修改
- GitHub Pages: 修改仓库名

### Q: 部署是免费的吗？
A: 是的！以上三种方法都完全免费。

### Q: 如何更新网站内容？
A: 
1. 修改代码
2. 运行 `npm run build`
3. 重新拖拽 `dist` 文件夹到 Netlify（或重新运行部署命令）

### Q: 可以绑定自己的域名吗？
A: 可以！所有平台都支持自定义域名，在设置中添加即可。
