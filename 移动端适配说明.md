## 🔄 如何测试

### 方法1：浏览器开发者工具
1. 打开浏览器（Chrome/Edge）
2. 按 `F12` 打开开发者工具
3. 点击设备切换按钮（或按 `Ctrl + Shift + M`）
4. 选择移动设备（iPhone、Android等）
5. 刷新页面

### 方法2：调整浏览器窗口
1. 缩小浏览器窗口宽度
2. 当宽度 < 768px 时自动切换到移动端
3. 放大窗口宽度会自动切换回PC端

### 方法3：真实手机测试
1. 确保手机和电脑在同一WiFi
2. 在手机浏览器访问：`http://192.168.28.52:5173`
3. 点击车辆进入详情页

## 💡 提示

### PC端和移动端的区别

| 特性 | PC端 | 移动端 |
|------|------|--------|
| 布局 | 左右分栏 | 上下堆叠 |
| 图片 | 左侧固定 | 顶部轮播 |
| 利润计算器 | 右侧固定 | 无（简化） |
| Tab | 横向Tab | 固定Tab |
| 操作栏 | 卡片内 | 底部固定 |

### 为什么移动端简化了？

1. **屏幕空间有限**：移动端屏幕小，需要简化内容
2. **操作习惯不同**：移动端用户更习惯滑动和点击
3. **性能考虑**：移动端性能较弱，简化可提升流畅度
4. **核心功能优先**：保留最重要的功能，隐藏次要功能
